# Web Technology Demos

This repository contains the source code for my personal website and a collection of web technology demos.

[**Visit the live website**](https://syedabutalib.github.io/)

## About This Site

This repository contains a collection of web technology demos. It is built using Jekyll and hosted on GitHub Pages.

## Demos

This site includes the following technical demonstrations:

*   **Media Source Extensions (MSE):** Demos for playing back various video and audio formats using MSE. See [`mse.html`](https://syedabutalib.github.io/mse.html).
*   **WebRTC:** A demo showcasing live video preview, recording, flipping, and rotation using WebRTC. See [`flip.html`](https://syedabutalib.github.io/flip.html).
*   **Audio Bit Depth Test:** A page to test audio capture and playback at different bit depths. See [`bitdepth.html`](https://syedabutalib.github.io/bitdepth.html).
*   **Spatial Audio:** A demo of spatial audio using Web Audio API. See [`spatial_audio.html`](https://syedabutalib.github.io/spatial_audio.html).
*   **Opus Audio:** A demo for playing Opus audio. See [`opus.html`](https://syedabutalib.github.io/opus.html).
*   **VTT:** A demo for WebVTT subtitles. See [`vtt.html`](https://syedabutalib.github.io/vtt.html).

## Personal Projects
